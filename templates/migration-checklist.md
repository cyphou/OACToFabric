## Pre-Migration

- [ ] Asset inventoried by Discovery Agent
- [ ] Dependencies identified and documented
- [ ] Complexity score assigned
- [ ] Migration wave assigned
- [ ] Pre-migration baseline captured (row counts, screenshots, query results)

## Migration

- [ ] Migration plan generated by responsible agent
- [ ] Migration executed
- [ ] Artifacts committed to Git
- [ ] No critical errors during migration

## Post-Migration Validation

### Data Validation (Tables/Views)
- [ ] Row count matches source
- [ ] Null counts match per column
- [ ] Aggregate totals match (SUM, COUNT, AVG)
- [ ] Sample row comparison passed
- [ ] Data type mapping verified

### Semantic Model Validation (Models)
- [ ] All measures return correct results
- [ ] Relationships produce correct joins
- [ ] Hierarchies drill correctly
- [ ] Time intelligence functions work
- [ ] Calculated columns match source values

### Report Validation (Analyses/Dashboards)
- [ ] All visuals render correctly
- [ ] Slicers/filters function correctly
- [ ] Drillthrough navigation works
- [ ] Conditional formatting applied
- [ ] Visual comparison approved (screenshot review)

### Security Validation (Roles)
- [ ] RLS filters return correct data per user
- [ ] OLS hides correct columns/tables
- [ ] Negative test passed (restricted data not visible)
- [ ] Cross-report security consistent

### Performance Validation
- [ ] Report load time acceptable
- [ ] Query execution time acceptable
- [ ] Data refresh time acceptable
